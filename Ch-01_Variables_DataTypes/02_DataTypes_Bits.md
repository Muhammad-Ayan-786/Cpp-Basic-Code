# Binary
In computers, binary is the language of 0s and 1s that computers use to store data and perform all operations.

---

## Bits and Bytes
- 1 bit = a single digit (0 or 1)
- 8 bits = 1 byte

---

## Bytes in int data type
- int takes 4 bytes
- 1 byte = 8 bits
- 8 bits * 4 = 32 bits  
✅ int has 32 bits

---

## Data Types
- int → 4 bytes
- char → 1 byte
- float → 4 bytes
- bool → 1 byte
- double → 8 bytes
